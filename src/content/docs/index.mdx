## Features

<CardGrid stagger>
	<Card title="High Iteration Speeds" icon="seti:clock">
    The ECS system of WasserXR is highly dynamic. Code of any system or component can be recompiled at runtime and directly hot reloaded.
	</Card>
	<Card title="Use any Language" icon="seti:html">
    The ECS system allows you to write any system or component in the language that is compatible with the C-ABI. You just link against the main ECS library and you can freely code.
	</Card>
	<Card title="Easy and Maintainable" icon="open-book">
    WasserXR is made to have easy and modular code by design. Therefore, it is easy to learn for beginners while also being extensible for advanced users.
	</Card>
	<Card title="Ready to Build" icon="rocket">
    WasserXR already comes with a wide variety of components and systems that make your life a lot easier to build good XR applications.
	</Card>
	<Card title="Open Source" icon="github">
    The engine is fully open source so you can go in the code and change what you need
	</Card>
</CardGrid>
