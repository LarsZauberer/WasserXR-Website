This is the starting point on the guide how to get started with WasserXR. In the
first part we will discuss how to setup your project to get something running.

<Tabs syncKey="lang">
<TabItem label="C">
Create a basic C project like you normally would. You can create a main executable with the following code.

```c
#include "TheSeed/core/logging.h"
#include "TheSeed/core/utils.h"
#include "TheSeed/ecs/Scene.h"

int main() {
  // Create logging

  ts_logging_init(TS_LOG_DEBUG);
  ts_add_logger(ts_stdout_logger);

  ts_info("Logging Initialized!");

  // Create the ECS
  TS_Scene *scene = ts_create_scene();

  // Load standard library
  ts_load_plugin(scene, "path/to/standard_library.so");

  // Initialize some standard entities and components
  TS_Entity console = ts_add_entity(scene);
  ts_add_component(scene, console, "TS_Console");

  TS_Entity window = ts_add_entity(scene);
  ts_add_component(scene, window, "TS_Window");

  TS_Entity camera = ts_add_entity(scene);
  ts_add_component(scene, camera, "TS_Camera");
  ts_add_component(scene, camera, "TS_Transform");

  // Initialize some standard systems
  ts_add_system(scene, "ts_console_system", 100);

  ts_add_system(scene, "ts_window_pre_renderer", 50);
  ts_add_system(scene, "ts_window_post_renderer", 150);

  ts_add_system(scene, "ts_window_quiter", 200);

  ts_add_system(scene, "ts_window_reloader", 100);

  ts_add_system(scene, "ts_mesh_renderer", 100);

  // Main Game Loop
  while (ts_tick_scene(scene)) {
  }

  // Clean Up
  ts_destroy_scene(scene);
}
```

Now, you need to compile the project

<Tabs syncKey="build-c">
<TabItem label="GCC">
To compile the project with [GCC](https://en.wikipedia.org/wiki/GNU_Compiler_Collection) you can run the following command

```bash
gcc -Ipath/to/includes -Lpath/to/WasserXR-Library -lwasserxr_ecs -o main main.c
```

</TabItem>

<TabItem label="CMake">
To compile the project with [CMake](https://cmake.org/) you can use something like the following

```cmake
cmake_minimum_required(VERSION 3.10)
project(MyProject LANGUAGES C)

add_executable(main main.c)
target_include_directories(main PRIVATE path/to/WasserXR-include)
target_link_libraries(main PRIVATE wasserxr_ecs)
```

You can then run

```bash
cmake -S . -B build

cmake --build build
```

</TabItem>

</Tabs>

</TabItem>
</Tabs>

## Result

When everything worked, you should see a black Window without anything
happening. This is what we want. This is the empty canvas where the game will be
created.
