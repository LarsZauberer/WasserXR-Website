This is the starting point on the guide how to get started with WasserXR. In the
first part we will discuss how to setup your project to get something running.

<Tabs syncKey="lang">
<TabItem label="C">

Create a basic C project like you normally would. You can create a main
executable with the following code.

```c
#include "WasserXR/ecs/logging.h"
#include "WasserXR/ecs/utils.h"
#include "WasserXR/ecs/Scene.h"

int main() {
  // Create logging
  wxr_logging_init(WXR_LOG_DEBUG);
  wxr_add_logger(wxr_stdout_logger);

  wxr_info("Logging Initialized!");

  // Create the ECS
  WXR_Scene *scene = wxr_create_scene();

  // Load standard library
  wxr_load_plugin(scene, "path/to/standard_library.so");

  // Initialize some standard entities and components
  WXR_Entity console = wxr_add_entity(scene);
  wxr_add_component(scene, console, "WXR_Console");

  WXR_Entity window = wxr_add_entity(scene);
  wxr_add_component(scene, window, "WXR_Window");

  WXR_Entity camera = wxr_add_entity(scene);
  wxr_add_component(scene, camera, "WXR_Camera");
  wxr_add_component(scene, camera, "WXR_Transform");

  // Initialize some standard systems
  wxr_add_system(scene, "wxr_console_system", 100);

  wxr_add_system(scene, "wxr_window_pre_renderer", 50);
  wxr_add_system(scene, "wxr_window_post_renderer", 150);

  wxr_add_system(scene, "wxr_window_quiter", 200);

  wxr_add_system(scene, "wxr_window_reloader", 100);

  wxr_add_system(scene, "wxr_mesh_renderer", 100);

  // Main Game Loop
  while (wxr_tick_scene(scene)) {
  }

  // Clean Up
  wxr_destroy_scene(scene);
}
```

Now, you need to compile the project

<Tabs syncKey="build-c">
<TabItem label="GCC">

To compile the project with
[GCC](https://en.wikipedia.org/wiki/GNU_Compiler_Collection) you can run the
following command

```bash
gcc -Ipath/to/includes -Lpath/to/WasserXR-Library -lwasserxr_ecs -o main main.c
```

</TabItem>

<TabItem label="CMake">

To compile the project with [CMake](https://cmake.org/) you can use something
like the following

```cmake
cmake_minimum_required(VERSION 3.10)
project(MyProject LANGUAGES C)

add_executable(main main.c)
target_include_directories(main PRIVATE path/to/WasserXR-include)
target_link_libraries(main PRIVATE wasserxr_ecs)
```

You can then run

```bash
cmake -S . -B build

cmake --build build
```

</TabItem>

## Result

When everything worked, you should see a black Window without anything
happening. This is what we want. This is the empty canvas where the game will be
created.
