## Function

The renderer is the main tick function of the OpenGL rendering system. It will
render a single frame.

Furthermore, if there doesn't exist an OpenGL context and Window, it will create
the following [resources](/getting_started/ecs/resources):

- `render_window` (Type:
  `(glium::winit::Window, glium::backend::glutin::Display<glium::glutin::surface::WindowSurface>)`
  (Rust))
