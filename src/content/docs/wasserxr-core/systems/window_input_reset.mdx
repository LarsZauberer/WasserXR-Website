| Purpose              | Resets the Window Events Resource |

## Function

This system is responsible to clear all the window events in the `window_events`
[resource](/getting_started/ecs/resource) that have been created before by the
[window_input_read](/wasserxr-core/systems/window_input_read) system.

The main reason this system exists is to make the `window_events`
[resource](/getting_started/ecs/resource) not always grow infinitely large.

<Aside type="caution" title="Priority" icon="warning">

This system should definitively come after the
[window_input_read](/wasserxr-core/systems/window_input_read) system and after
every system that wants to read the window events.

</Aside>
