| Type    | Component                                                     |
| ------- | ------------------------------------------------------------- |
| Name    | `Camera`                                                      |
| Purpose | Provides properties about the camera options for the renderer |

## Fields

| Field  | Type  | Default | Description                           | Virtual | Accessible | Mutable | Serializable | Deserializable |
| ------ | ----- | ------- | ------------------------------------- | ------- | ---------- | ------- | ------------ | -------------- |
| `fov`  | `f32` | `90.0`  | The field of view (FOV) of the camera | NO      | YES        | YES     | YES          | YES            |
| `near` | `f32` | `0.1`   | Near clipping plane of the camera     | NO      | YES        | YES     | YES          | YES            |
| `far`  | `f32` | `1000.0` | Far clipping plane of the camera      | NO      | YES        | YES     | YES          | YES            |
