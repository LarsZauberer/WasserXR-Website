## Allowed Uses

You may:

- refer to WasserXR in tutorials, videos, articles, reviews, benchmarks, and
  discussions
- state that your project, plugin, or tool is "for WasserXR" or "compatible with
  WasserXR"
- create plugins, integrations, tools, language bindings, and extensions for
  WasserXR
- use the WasserXR name in descriptive text
- reference WasserXR in package names when it is clear that the project is
  unofficial.

Examples:

"Physics Toolkit for WasserXR" "WasserXR Rust Bindings" "Compatible with
WasserXR"

## Not Allowed

You may not:

- imply that your project is officially affiliated with or endorsed by WasserXR
  when it is not

- distribute modified versions of WasserXR while representing them as official
  releases
- use the WasserXR branding for scams, malware, or illegal activity
- register domains, organization names, or social media accounts that
  impersonate the official WasserXR project.

Examples of prohibited names:

"Official WasserXR Pro" "WasserXR Corporation" "WasserXR Official Runtime"
